## System Requirements

- PHP 7.4 or higher
- MySQL 5.7+ or MariaDB 10.3+
- Apache 2.4+ with mod_rewrite
- Required PHP extensions: PDO, cURL, JSON, Session, OpenSSL, Mbstring

## Features

- **Multi-Provider AI Integration**: OpenAI, Claude, Gemini
- **Intelligent Categorization**: Automatic ticket classification
- **Smart Response Generation**: Context-aware AI responses
- **Ticket Tracking**: Comprehensive tracking system
- **Analytics Dashboard**: Detailed metrics and reporting
- **Security Focused**: CSRF protection, input validation, secure sessions
